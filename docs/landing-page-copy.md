## Features

### Memory that evolves
pai doesn't forget. It builds beliefs about you over time — reinforcing what's consistent, contradicting what's changed, decaying what's stale. Ask it something from three months ago. It remembers.

### Learns from the web
Point it at any URL and it learns the content. Documentation sites, articles, research papers. Then ask questions about what it's read — with semantic search and re-ranking, not just keyword matching.

### Web search with citations
Real-time web search powered by SearXNG. Every claim backed by an inline source link. Like Perplexity, but private and self-hosted.

### Deep research
Ask it to research a topic and it spawns specialized agents — flight prices, stock analysis, crypto, news — working in parallel to deliver a comprehensive report.

### Tasks & scheduling
Manage tasks with AI prioritization. Set up recurring research — daily news briefings, weekly market summaries — delivered to your inbox automatically.

### Code execution
Run Python and JavaScript in a sandboxed environment. Data analysis, charts, calculations — right in the conversation.

### Works everywhere
Web UI, Telegram bot, CLI, or integrate with Claude Code / Cursor / Windsurf via the built-in MCP server with 19 tools.
